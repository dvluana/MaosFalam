<!-- Backend API contract (POST /api/credits/purchase) -->
Request body (Zod schema):
```typescript
{
  pack_type: string;       // "avulsa" | "dupla" | "roda" | "tsara"
  cpf?: string;            // 11-14 chars, optional
  reading_id?: string;     // UUID, optional (for upsell flow)
}
```

Response (200):

```typescript
{
  checkout_url: string;
}
```

Errors: 400 (invalid data), 401 (not authenticated), 429 (rate limit), 500 (server error)

```typescript
{
  clerk_user_id: string;
  name: string;
  email: string;
  cpf: string | null; // null = needs CPF collection
  phone: string | null;
}
```
