Create a reusable SVG component showing an open palm outline (5 fingers, wrist, palm body). Per CAM-04 and docs/maodominante.md section 2.4.

Props interface:

```typescript
interface HandOutlineSVGProps {
  dominantHand: "right" | "left";
  size?: number; // default 200
  className?: string;
}
```

SVG design rules:

- Draw a RIGHT hand palm (viewer's perspective: thumb on left side, pinky on right)
- Stroke color: gold-dim (#7a6832) at opacity 0.15, strokeWidth 1.5
- Fill: none (outline only)
- When dominantHand="left", apply style transform: scaleX(-1) to flip horizontally
- When dominantHand="right", no transform
- viewBox sized to fit palm proportions (roughly 200x280)
- The SVG should show: wrist base, palm body, 5 fingers with natural curvature, spread apart as if showing the palm
- Use path elements with bezier curves for natural hand shape
- Add role="img" and aria-label="Guia de posicionamento da mao {direita/esquerda}" (per docs/maodominante.md a11y section)
- Export default
- "use client" directive

Do NOT use emojis. Do NOT reference energias/vibracoes/universo.
